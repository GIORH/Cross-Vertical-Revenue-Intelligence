# 📈 Enterprise Sales Performance Engine

## 🚀 Project Overview
A mission-critical data ecosystem designed to unify, normalize, and analyze sales performance across diverse business verticals: **Mobility, Smart Home Security, FFH, and Health.**

This engine automates the journey from raw sales leads to final executive revenue insights, providing a scalable infrastructure for commission accuracy and sales trend analysis.

---

## 🏗️ Data Architecture
The pipeline is structured into three specialized layers:
1. **Normalization Layer:** SQL procedures that standardize different product units (e.g., fiber subscriptions vs. mobile devices) into a unified Sales Metric.
2. **Incentive Engine:** Advanced BigQuery logic to calculate tiered commissions based on dynamic targets.
3. **Executive BI Layer:** Optimized views for **Looker Studio** providing a 360° view of global sales health.

---

## 🛠️ Key Technical Implementations
* **Multi-Vertical Consolidation:** Handling disparate CRM data schemas.
* **Defensive SQL:** High-integrity scripts using `SAFE_CAST` and `REGEXP` for data cleaning.
* **Apps Script Integration:** Automated intake systems for real-time sales reporting.
